Project Description:
	A web page to display a list of movies.
	Along with the movies it displays some details like Movie title, Movie Image, release date and actors.
	When user clicks on the Movie image, it pops a window to display trailer.
	
Files in the Project:
1. fresh_tomatoes.html is a dynamically created file which when opened in a browser displays the Movie website.
2. Run fresh_tomatoes.py to get the fresh_tomatoes.html file.
3. media.py file contains the movie object structure.
4. entertainment_center1.py creates movie object.
